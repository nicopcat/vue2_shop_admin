# vue2_shop_admin
目标是一个基本的电商后台系统。
## 启动 Project
安装依赖：
```
npm install
```

预览：
```
npm run serve
```
## 后台数据
若要体验该应用，需要先运行phpStudy和数据库才能发送请求。

### 步骤
1. 安装 phpStudy， 导入准备好的 sql 文件  
2. cd 准备好的文件夹`vue_api_server`，终端中输入命令安装项目依赖包：`npm install`  
3. 运行 powershell/cmd，输入`node ./app.js`指令，运行数据库，此时方可登录  
4. 用户名：admin  
5. 密码：随意
